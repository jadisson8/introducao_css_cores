Questão 01

Data de entrega: 23 de ago., 14:00

Crie uma página html como se fosse um blog com as seguintes instruções:

  1. A página deve ter uma tag h1 com o título ”Exercício de HTML e CSS 01 – Seu nome”;
  2. Cada postagem ou seção deve representar as questões de 02 a 05 abaixo. O título deve ser um texto em um h2 com o número da questão. Ex: ”Questão 02”;
  3. Abaixo de cada h2 deve existir uma tag <hr> seguido de parágrafo com uma data;
  4. Cada postagem deve ter um pequeno parágrafo explicando o que está sendo pedido na questão e em seguida a sua resolução.
